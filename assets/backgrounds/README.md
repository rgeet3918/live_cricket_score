# Background Images

Please add the following background images to this folder:

1. **nature_path.jpg** - Autumn forest path with winding trail through colorful trees
2. **golden_sparkle.jpg** - Golden/yellow abstract background with bokeh lights and glitter effect
3. **neon_smoke.jpg** - Dark background with neon blue and pink smoke/fog effects
4. **floral_wall.jpg** - Wooden plank wall with pink and yellow flowers cascading down

## Image Requirements:
- Format: JPG or PNG
- Resolution: 1080x1920 or higher (recommended)
- File names must match exactly:
  - nature_path.jpg
  - golden_sparkle.jpg
  - neon_smoke.jpg
  - floral_wall.jpg

## How to Add:
1. Save your images with the correct names
2. Copy them to this directory: `/assets/backgrounds/`
3. Run `flutter pub get` to refresh assets
4. Restart the app to see the new backgrounds
